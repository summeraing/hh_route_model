# Release notes: evidence-atlas expanded computation

This update adds the route-resolved evolutionary evidence-atlas computational package.

New public materials:

- `code/evidence_atlas/`: key calculation scripts for probabilistic mixture, predictive/permutation evidence, information theory, feature ablation, unsupervised recovery, block-level MDL, hierarchical resampling, Shapley and adversarial diagnostics.
- `data/evidence_atlas/Source_Data_route_resolved_evidence_atlas_complete.xlsx`: consolidated processed Source Data workbook.
- `data/evidence_atlas/block_mdl_model_evidence_v1.xlsx`: block-level MDL and held-out model-evidence results.
- `data/evidence_atlas/unsupervised_route_recovery_v1.xlsx`: route-independent correspondence, modularity and clustering diagnostics.
- Additional processed workbooks for predictive cross-validation, permutation nulls, feature ablation, information theory, hierarchical resampling, evidence accumulation and adversarial analyses.

This repository intentionally excludes manuscript files, cover letters, Supplementary Information documents, submission figures, image-design files, slides and editorial planning materials.

The public DOI is retained as the repository-level Zenodo DOI unless a new version-specific DOI is minted.
