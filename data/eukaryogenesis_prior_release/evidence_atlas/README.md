# Route-resolved evidence-atlas processed data

This folder contains processed numerical workbooks for the route-resolved evolutionary evidence-atlas analysis.

Included files:

- `Source_Data_route_resolved_evidence_atlas_complete.xlsx`: consolidated Source Data workbook with curated evidence tables and appended computational evidence tables.
- `evidence_units_flat.csv`: flat 1,522-row processed evidence-unit table used by the standalone route-recovery and block-MDL scripts.
- `RouteB_computational_upgrades_v1.xlsx`: probabilistic donor-role mixture and rule-based audit-label cross-check.
- `expanded_computational_evidence_v1.xlsx`: model-family, posterior, stress-test and network-concentration diagnostics.
- `predictive_permutation_evidence_v1.xlsx`: leave-source/layer predictive cross-validation and stratified permutation nulls.
- `hierarchical_resampling_shapley_v1.xlsx`: clustered resampling and source/layer Shapley diagnostics.
- `evidence_accumulation_forecast_v1.xlsx`: evidence-layer accumulation and future-source adversarial-boundary forecasts.
- `routeB_information_theory_evidence.xlsx`: source-aware mutual-information diagnostics.
- `routeB_predictive_feature_ablation.xlsx`: supervised feature-ablation diagnostics.
- `unsupervised_route_recovery_v1.xlsx`: correspondence, modularity and clustering route-recovery diagnostics.
- `block_mdl_model_evidence_v1.xlsx`: block-level MDL and held-out model-evidence diagnostics.
- `adversarial_counterfactual_v1.xlsx`: matrix-level adversarial route-erasure diagnostics.
- `row_level_adversarial_v1.xlsx`: row-level adversarial influence diagnostics.

These are processed reproducibility tables. Manuscript files, cover letters, Supplementary Information documents, final submission figures, slides and editorial planning materials are intentionally excluded from this public repository.
