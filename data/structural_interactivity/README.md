# AF3 structural-interactivity support package

This folder contains a compact archive supporting the structural-interactivity layer of the route-resolved eukaryogenesis analysis.

## Archive contents

The archive contains:

- 19 representative model CIF files.
- 19 AF3 summary confidence JSON outputs.
- 19 ranking-score CSV files.
- Integrated interface-metric tables for the figure/source-data layer.
- A README describing interpretation boundaries.

## Interpretation boundary

These files support a gate-calibrated structural-feasibility screen. AF3 confidence scores, buried surface area and contact counts are interpreted together. The archive is not a direct reconstruction of ancient binding interfaces and does not include proprietary AF3 model parameters.
