# Core metrics data

This folder contains compact derived numerical tables used by `code/reproduce_core_metrics.py`.

These files are enough to verify the locked headline calculations reported in the manuscript, but they are not the full evidence-unit source-data release. The full workbook remains part of the journal Source Data package.

Files:

- `route_map_specificity_adjudicated.csv`
- `frozen_null_boundary.csv`
- `dependency_collapse.csv`
- `agreement_summary.csv`
- `route_support_summary.csv`
- `adjudication_summary.csv`
