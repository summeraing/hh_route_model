# Processed SOX genome-atlas data

This directory contains the processed tables required to inspect and validate the genome-atlas claims. It includes candidate QC, six family trees, fixed and ModelFinder reconciliation summaries, matched mobile/composition controls, route and continuous-map outputs, and cross-layer family sensitivities.

The route-input table is `13_atlas_evidence_fixed/sox_atlas_evidence_units.csv`. Its 3,658 route-eligible rows are traceable source-by-layer-by-operon-role entries, not independent biological replicates; dependency-group identifiers and collapsed outputs are provided.

Large public source files, raw RefSeq genomes and GTDB database files are not redistributed. The complete compact reproducibility archive is attached to the corresponding GitHub release and archived with Zenodo.
